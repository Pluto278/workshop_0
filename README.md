# workshop_0
https://pluto278.github.io/workshop_2/
This p5.js sketch creates a dynamic and colorful animation featuring random 2D shapes (circles, rectangles, and triangles) that are generated and drawn on the canvas. The shapes are randomly positioned, sized, and colored, adding variety to the visual display.

Key features include:
	•	2D Primitive Shapes: The sketch uses ellipse, rect, and triangle to create a variety of geometric shapes.
	•	Randomized Attributes: Positions, sizes, colors, and even the types of shapes are determined randomly, ensuring a unique animation each time it runs.
	•	Interactive Behavior: Clicking the mouse clears all the shapes on the canvas, allowing the user to reset the animation.
	•	Trail Effect: By setting a semi-transparent background, a fading trail effect is created, giving the animation a smooth and flowing appearance.
	•	Performance Optimization: The sketch limits the number of shapes to the most recent 50 to prevent excessive memory usage.

This sketch also experiments with the frameRate() function from the p5.js documentation to control the speed at which shapes are added to the canvas.
